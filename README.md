## Front

### 기대효과
주문 배달 서비스 구현을 통해 자바스크립트 / 리액트 복습, api 연동 등을 다시 한번 학습합니다. <br/>
UI 디자인 구성은 Figma를 이용했고 디자인 구현은 Tailwind CSS와 Emotion을 사용했습니다.
